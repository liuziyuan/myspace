## 🐍 Python

### 📥 安装方式

#### uv（推荐）

```bash
# Homebrew 安装
brew install uv
```

### 🔄 更新方式

```bash
# Homebrew 更新
brew upgrade uv
```

### 🗑️ 卸载方式

```bash
# Homebrew 卸载
brew uninstall uv
```

### 💡 项目中使用

```bash
# 创建新项目
uv init my-project
cd my-project

# 创建虚拟环境
uv venv                        # 使用默认版本
uv venv --python 3.12          # 指定 Python 版本
uv venv -p 3.11                # 简写形式

# 安装依赖（两种方式）
uv pip install requests          # 传统方式
uv add requests                  # 推荐方式，自动更新 pyproject.toml

# 运行脚本
uv run python main.py

# Python 版本管理
uv python install 3.12          # 安装特定版本
uv python list                  # 查看可用版本
```

## 💚 Node.js

### 📥 安装方式

```bash
# 安装 Volta
brew install volta
```

### 🔄 更新方式

```bash
# 更新 Volta
brew upgrade volta

# 安装新版本 Node
volta install node@20

# 查看已安装的版本
volta list all
```

### 🗑️ 卸载方式

```bash
# Homebrew 卸载
brew uninstall volta
```

### 💡 项目中使用

```bash
# 安装 Node 版本
volta install node@20

# 项目中使用特定版本
cd my-project
volta pin node@18

# 安装包管理器
volta install npm@latest
volta install pnpm@latest
volta install yarn@latest

# 项目固定包管理器版本
volta pin yarn@1.22.0

# 运行脚本（自动使用项目指定版本）
npm run dev
```

## ☕ Java

### 📥 安装方式

```bash
# 官方脚本安装
curl -s "https://get.sdkman.io" | bash

# 重新加载 shell 配置
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

### 🔄 更新方式

```bash
# 更新 SDKMAN 本身
sdk selfupdate

# 安装新版本 Java
sdk install java 21.0.1-tem

# 查看已安装的版本
sdk list java | grep installed
```

### 🗑️ 卸载方式

```bash
# 卸载特定 Java 版本
sdk uninstall java 21.0.1-tem

# 完全卸载 SDKMAN
rm -rf ~/.sdkman
# 并删除 shell 配置文件中的相关行
```

### 💡 项目中使用

```bash
# 查看可用的 Java 版本
sdk list java

# 安装特定版本（推荐 Temurin/Eclipse Adoptium）
sdk install java 21.0.1-tem     # Java 21 LTS
sdk install java 17.0.9-tem     # Java 17 LTS
sdk install java 11.0.21-tem    # Java 11 LTS

# 临时切换版本（当前终端会话）
sdk use java 17.0.9-tem

# 设置默认版本
sdk default java 21.0.1-tem

# 项目中使用特定版本（在项目根目录）
sdk use java 17.0.9-tem

# 查看当前使用的版本
sdk current java

# 安装其他工具（Gradle、Maven 等）
sdk install gradle 8.5
sdk install maven 3.9.6
```
